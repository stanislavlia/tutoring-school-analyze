## **ТЗ: Система анализа успеваемости городских школ**

### **Исходные данные:**
- **5 школ** (Школа №1, №2, №3, №4, №5)
- **Структура классов в каждой школе:**
  - 8-9 классы: по 3-4 буквы (8А, 8Б, 8В, 8Г и т.д.)
  - 10-11 классы: по 2 буквы (10А, 10Б и т.д.)
- **В каждом файле класса:**
  - ФИО ученика (15-30 учеников случайно)
  - Оценки по предметам: Математика, Русский язык, Физика, Химия, Биология, История
- **Формат:** `/школа_N/класс_буква.xlsx` (например, `/школа_1/8А.xlsx`)

---

### **Задачи программы:**

**1. Сводка по каждой школе** (`школа_N_сводка.xlsx`):
   - Для каждого класса: средний балл по всем предметам
   - Рейтинг классов внутри школы
   - Общая статистика школы

**2. Сводка по параллелям города** (`параллель_8_город.xlsx`, `параллель_9_город.xlsx` и т.д.):
   - Средние баллы всех классов данной параллели из всех школ
   - Рейтинг классов в параллели по городу
   - Определение лучшего класса параллели

**3. Топ-10 выпускников города** (`топ_11_класс_город.xlsx`):
   - По каждому предмету отдельно: топ-10 учеников 11-х классов всего города
   - Формат: ФИО, Школа, Класс, Оценка

---

**Дополнительно:**
- Цветовое выделение: отличники (зеленый), хорошисты (желтый), остальные (белый)
- Графики для наглядности (опционально)

**Оценка времени:** ~3 часа
